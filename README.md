## 🛠️ The Terminal Core
This 50MB+ heavyweight PWA (Progressive Web App) is engineered for zero-latency financial tracking and milestone management.

### 💰 Wealth Intelligence & Ledger
* **Investable Surplus Tracking:** Real-time inflow/outflow monitoring with 10-year wealth forecasting (12% CAGR logic).
* **AI Spending Anomaly Detection:** Built-in alerts for high-outflow detection to maintain capital discipline.
* **Persistence:** Powered by **Firebase Firestore** with IndexedDB for 100% offline functionality.

### 🎓 ACCA Milestone Tracker
* **Sinking Fund Engine:** Live currency conversion (GBP/USD/INR) using real-time Forex APIs to calculate exact exam capital requirements.
* **Progress Mapping:** Visual milestone tracking for all 13 ACCA papers (Applied Knowledge, Skills, and Strategic Professional).
* **Market Value Projection:** Dynamic career salary forecasting based on paper completion status.

### 🛡️ Elite Security
* **Terminal Lock:** 256-bit encrypted authentication with a 4-digit biometric-style PIN gate.
* **Anti-Flicker Launch:** Engineered with a custom DOM-guard to ensure the dashboard never leaks data during PWA initialization.
* **Identity Sync:** Fully encrypted user profiles including DOB and Gender synchronization.
